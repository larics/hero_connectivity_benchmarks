# Benchmark world examples for Benchmarking Connectivity and Battery-aware algorithms with Crazyflies 

This work accompanies sim2real framework for benchmarking connectivity and battery aware algorithms with Crazyflie UAVs. It is supporting both simulation using CrazySim + Gazebo and real hardware experiments. 

# Worlds 

Icuas25/worlds contains: 
- Small world 10 × 10m^2, 15 obstacles, 8 targets
- Medium-sized world: 15 × 15 m^2, 25 obstacles, 15 targets
- Large world: 20 × 20 m^2, 45 obstacles, 25 targets


# Environment 
The environment for setting up these worlds is given in repository: https://github.com/larics/hero_crazyflies_benchmarking.git
